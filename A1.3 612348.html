3. Se utiliz la librera statsmodels para ajustar un modelo de regresin lineal mltiple mediante la funcin OLS.
En este bloque se definieron las variables predictoras: frecuencia, ngulo, longitud, velocidad y espesor, y la variable respuesta: presin.
Posteriormente, se incluy una constante con add_constant para considerar el intercepto en el modelo y se imprimi el resumen completo de resultados con summary(), as como los p-values de cada coeficiente.

import statsmodels.api as sm

Xcols = ["frecuencia","angulo","longitud","velocidad","espesor"]
ycol = "presion"

X_train = train[Xcols]
y_train = train[ycol]

X_test = test[Xcols]
y_test = test[ycol]

X_train_sm = sm.add_constant(X_train)
X_test_sm = sm.add_constant(X_test)

results = sm.OLS(y_train, X_train_sm).fit()

print(results.summary())
print("P-values por variable (incluye const):")
print(results.pvalues)

5. Se calcularon dos mtricas de desempeo del modelo en los conjuntos de entrenamiento y prueba:

  • RSE (Residual Standard Error): se obtuvo a partir de la raz cuadrada del error cuadrtico medio.
  • R (Coeficiente de determinacin): mide la proporcin de la variabilidad explicada por el modelo.
    De esta forma, se imprimieron los valores de RSE y R para ambos conjuntos, lo que permiti comparar la capacidad de ajuste en entrenamiento y validacin.
In[18]:
from sklearn.metrics import r2_score, mean_squared_error
import math

RSE_train = math.sqrt(results.scale)
R2_train = results.rsquared

y_pred_test = results.predict(X_test_sm)
RSE_test = math.sqrt(mean_squared_error(y_test, y_pred_test))
R2_test = r2_score(y_test, y_pred_test)

print("RSE (train):", RSE_train)
print("R  (train):", R2_train)
print("RSE (test):", RSE_test)
print("R  (test):", R2_test)
